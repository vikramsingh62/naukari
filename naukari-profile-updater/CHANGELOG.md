## [1.0.0] - 2024-01-15

### Added
- Initial release
- Automated login to Naukari.com
- Profile punctuation toggle feature (add/remove full stop)
- Scheduler for daily execution at 10:30 AM
- GCP Compute Engine deployment guide (free tier)
- Local scheduler support
- Comprehensive logging and execution history
- Debug mode with screenshots
- Environment-based configuration management
- Support for .env and config.json configuration
- Unit tests and integration test templates
- Production-ready error handling
- Security best practices (credential management)
- GitHub Actions CI/CD pipeline
- Documentation and deployment guides
- MIT License
